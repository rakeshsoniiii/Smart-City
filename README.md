# JanAI Prototype

JanAI is a people-centered smart city intelligence prototype for India. This demo includes three working modules:

- Education recommendation
- Hospital and health support
- Safe route planning

## Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python standard-library HTTP server
- Data: Local JSON demo datasets

## Run locally

1. Start the app:

```powershell
python backend\main.py
```

2. Open:

```txt
http://127.0.0.1:8000
```

## Demo Inputs

- Education: `Delhi`, `100000`, `Computer Science`
- Healthcare: `Delhi`, `Chest pain and difficulty breathing`
- Safety: `Delhi` to `Noida`, `Evening`
